# Enterprise Network Project - CCNA

## Project Overview
Complete 3-building enterprise network
designed and configured using
Cisco Packet Tracer.

## Topology
- 2 ISP Routers
- 2 Core Routers (R1, R2)
- 2 Multilayer Switches (MSW1, MSW2)
- 6 Access Switches
- 3 Servers (DNS/NTP, DHCP/Syslog, HTTP/FTP)
- 5 Departments with 500+ users

## Technologies Configured
- OSPF Dynamic Routing
- HSRP Gateway Redundancy
- NAT/PAT Internet Access
- VLANs and Trunking (13 VLANs)
- DHCP with ip helper-address
- SSH Remote Access
- ACLs Department Isolation
- Rapid PVST+ STP
- WPA2 Wireless Security
- Port Security
- NTP Time Synchronization
- Syslog Centralized Logging
